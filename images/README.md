# 画像ファイル

このディレクトリに以下の画像を配置してください：

## 必要なファイル

### app-mockup.png
- アプリのスクリーンショットまたはモックアップ画像
- 推奨サイズ: 350px × 700px程度
- 形式: PNG

### app-store-badge.svg
- App Storeダウンロードバッジ
- Apple公式サイトからダウンロード: https://developer.apple.com/app-store/marketing/guidelines/
- 形式: SVG（推奨）またはPNG

## オプション

### favicon.ico / favicon.png
- ブラウザタブに表示されるアイコン
- サイズ: 32x32 または 16x16

### og-image.png
- SNSシェア時に表示される画像（OGP画像）
- 推奨サイズ: 1200px × 630px
